Local Network (LAN) mode — communicates with your Marstek battery over your home WiFi using the device's built-in Open API (UDP JSON-RPC, default port 30000). This works from anywhere on the same network, with no Bluetooth required and no range limit. Because browsers cannot send raw UDP packets, a small local bridge script must be running on this computer (step 1 below).

Limitations: the Open API exposes a smaller set of data than BLE — energy totals, battery state, solar input, WiFi status, and operating mode. Detailed diagnostics (BMS cell voltages, error logs, event history, firmware update) are only available over Bluetooth. The Local API must first be enabled on the device via the Marstek app or using BLE command 0x28 from this tool.

Full command reference: Marstek Device Open API (PDF).
